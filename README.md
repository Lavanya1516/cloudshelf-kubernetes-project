# CloudShelf Kubernetes Project

## Project Overview

CloudShelf is a production-style Kubernetes deployment project demonstrating core Kubernetes concepts such as:

- Namespace
- ConfigMap
- Secret
- Persistent Volume
- Persistent Volume Claim
- PostgreSQL
- NGINX
- Services
- Ingress
- Horizontal Pod Autoscaler
- Resource Management
- Network Policies
